开Performance Profile的时候找不到pdb

System.Private.CoreLib.ni.pdb 找不到这个

ni是native image

把这个禁掉

### ✔ 打开你的项目 → 右键 → **Properties（属性）**

### ✔ 选择 **Debug** 面板

### ✔ 找到 **Environment variables（环境变量）**

在里面添加这两个：

```
COMPlus_ReadyToRun=0
COMPlus_ZapDisable=1
```
